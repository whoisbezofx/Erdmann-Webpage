# H.G. Erdmann Sanitär- und Heizungsbau GmbH – Website

Moderne Webseite für H.G. Erdmann Sanitär- und Heizungsbau GmbH, Hagenburg.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **shadcn/ui**
- **Lucide Icons**

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
